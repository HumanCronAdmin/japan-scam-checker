## Why This Tool?

Tourist traps and scams target foreigners in Japan. Search by name or browse by category to check if a business has been reported.

## Features

- **Search by name** — Enter a business name and get an instant verdict
- **8 category filters** — Bar & Nightlife, Street, Tour, Taxi, Ticket, Hotel, Shopping, and more
- **3-level verdict** — Reported Scam (red), Verified Safe (green), Not in Database (yellow)
- **Common scam guide** — Learn about the most common scam types in Japan
- **Report a scam** — Submit new reports via Google Forms

## How It Works

1. Enter a business name or pick a category
2. See the verdict — Reported, Safe, or Unknown
3. Report new scams to help other travelers

> **Try it now:** [https://humancronadmin.github.io/japan-scam-checker/](https://humancronadmin.github.io/japan-scam-checker/)

## Data Sources

Scam reports are sourced from community submissions and publicly available information.

## Privacy

- All searches run **in your browser** against a local database — no data is sent to any server
- No cookies, no account required
- Works offline after first visit (PWA)
